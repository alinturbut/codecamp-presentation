What does JHipster bring to the table?

Let's dive deeper into JHipster features before getting our hands "dirty"

JHipster brings a simple way of generating a full webapp project in a matter of minutes.

yo jhipster

...and after deiciding on the technologies, you have a working webapp locally.

JDL

JHipster Domain Language helps you generate the domain entites of the webapp.

http://www.jhipster.tech/jdl/

JDL Studio

JDL Studio makes entity generation even more simple. It generates entities from simple json-like files with have certain keyboards in it. entity (generates a domain class with the specified name) relationship (add a relationship to the specified domain entities)

JHipster UML

http://www.jhipster.tech/jhipster-uml/

JHipster UML is a tool that generates domain entities from UML diagrams. It is a project in itself. There is even a JHipster IDE (Eclipse with custom plugin). Note: I am not going to talk a lot about all the features, because it would take too much time and time is a valuable thing (no pun intended).
